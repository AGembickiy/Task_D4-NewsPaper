# Task_D4-NewsPaper
Утанавливаем виртуальное окружение
python -m venv venv
Активируем его 
.\venv\Scripts\activate
Обновляем пакеты
python -m pip install --upgrade pip
Устанавливаем пакеты
pip install -r requirements
Устанавливаем Django filtr
pip install Django filtr
Не забываем подключить в INSTALLED_APPS
Выполняем команду
pip install -r requirements
python manage.py migrate
Создаём супер пользователя
python manage.py createsuperuser
Выполняем команды из файла
console_commandes
Далее сверяем с файлом ТЗ.docx
